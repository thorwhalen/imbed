imbed.components.segmentation

Segmentation functions to get text segments.

This includes tools to extract segments from text data, but also transform standard segments sources into a format ready to be used with the imbed library.

Convert various JSON-friendly formats to segments.

JSON-friendly formats we handle here are: - str (a single string) - list (and and iterable, of strings) - dict (whose values are strings)
