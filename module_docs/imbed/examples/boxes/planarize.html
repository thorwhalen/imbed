Update a dictionary with another dictionary, with more control over the update.

For example, if overwrite is False, then only keys that are not already in the dictionary will be added. If overwrite is True, then all keys will be added. If overwrite is a Callable, then it will be called with the key to determine whether to overwrite the key or not.
